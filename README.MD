# 1. 项目说明
B/S架构设计的软件。
功能：扫描本机指定目录视频图片作为为一个简易的自建NAS。

## 2. 注意 
1 运行server的环境要求有 ffmpeg （构建媒体库缩略图用）
2 JDK 17
3 本应用使用内嵌h2 db运行，最小化依赖设计
